Element Plus的el-checkbox组件选中效果在打印时不显示的解决思路

Posted at — Feb 16, 2023

公司项目其中有一项功能是在线填写&打印表格，在打印的时候遇到el-checkbox无法打印出选中的问题。

我要实现的表格如下：

打印效果如下：

经过用开发者工具排查应该是浏览器忽略了checkbox的背景颜色，导致checkbox选中的蓝色背景颜色无法显示，并且由于选中的checkbox中after伪元素的符号是白色，所以在打印时看起来是未选中的状态。

解决方法：

1.可以直接修改伪元素的符号颜色（这里我没做测试）

2.我的解决方法：

由于为了样式符合整体表格的风格，我直接修改了checkbox选中的边框颜色、背景颜色和伪元素：

table >>> .el-checkbox__inner {
  border-color:black;
}
table >>> .el-checkbox__input.is-checked .el-checkbox__inner {
  background: none;
  border-color:black;
}
table >>> .el-checkbox__input.is-checked .el-checkbox__inner::after {
  content: '\2714';  /* 设置选中后的✓符号*/
  transform:none;
  top: 5px;
  height: 0;
  left:1px;
}
